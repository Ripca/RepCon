## 🚀 Inicio Rápido

### Opción 1: Web (30 segundos)
```bash
cd web_project
python -m http.server 8000
# Abrir http://localhost:8000
```

### Opción 2: Python (2 minutos)
```bash
cd python_project
pip install -r requirements.txt
python app.py
# Abrir http://localhost:5000
```
